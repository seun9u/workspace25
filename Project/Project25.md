# 📝 자기계발 미션 트래커 프로젝트 계획서

---

## 1. 프로젝트 개요

- **프로젝트명**: HabitSpark (가제)
- **목표**: 사용자가 자기계발 습관을 설정하고, 매일의 실천 여부를 기록·시각화할 수 있는 웹 기반 앱 구현
- **개발 플랫폼**: Flutter Web
- **핵심 기능**:
  - 루틴 생성 및 반복 요일 설정
  - 달성 체크 UI (체크박스 or 스위치)
  - 달성률 시각화 (달력/그래프)
  - 연속 달성 보상 뱃지 시스템
  - 로컬(Hive) or 클라우드(Firebase) 저장소 연동
